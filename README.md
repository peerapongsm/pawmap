# PawMap · หมาแมวจร

Stray dog/cat sighting map app — report sightings on a map like Google Maps
incidents, see density, and export area summaries for municipalities.

**Status: pitch stage.** The client pitch (mockups + cost breakdown) lives in
[`docs/`](docs/) and is served via GitHub Pages. The app codebase (Expo React
Native + Supabase PostGIS) will live in this repo once development starts.

> "PawMap" is a working name and may change.
